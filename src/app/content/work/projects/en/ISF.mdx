## Overview

In this project, This is my first project as a structural designer. I've never done it before but I'm sure I can follow best practice.

## Key Features

- **Upload**: Seamlessly upload images and manage them.
- **Record Management**: Easily manage and customize your data columns, from material costs to suppliers.
- **Real-time Chat Support**: Get quick assistance through our chat feature.
- **User Management**: Control user roles and permissions effectively.
- **Dark/Light Mode Toggle**: Toggle between modes for a personalized experience.
- **Multi Language**: Support 4 languages in Application.

## Technologies Used (Front-End Only)

- **Material UI**: For save time to design component.
- **React and Next.js**: For building the front-end codebase with clean, reusable components.
- **Next-i18next and i18n**: For support multi language.
- **GitHub Actions**: For automating the pipeline and syncing design changes to the repository and use CICD to deploy APP.
