## 概要

このプロジェクトは古いソフトウェアです。 機能において、新規の商用またはサポートが何かを追加または編集する必要がある場合、学習曲線が必要になります。次に、React + ElectronJs を使用して Windows アプリで使用できるように移行します。

## 主な機能

- **UIUX is simple**: 古言語の UX と UI は使いにくくなりますが、新しい UX UI は使いやすいです
- **More than Lighten**: exeファイルでインストールし、サーバーのIPを設定するだけです

## 使用されるテクノロジー

- **React と Electron**: クリーンで再利用可能なcomponentとWindowsアプリへのビルド用の Electron を使用したフロントエンドコードベースの構築用
- **Figma**: デザインアセットとプロトタイプを作成および管理します
- **Styled-Components**: modular式のテーマ設定可能なアプローチでReact componentをスタイリングします
- **Material-UI**: componentの設計時間を節約するため
- **Miro**: ビジネスフローの再確認とエンドユーザーにとってのシンプルさ
- **Java と Springboots**: APIとバックエンド

## 課題と学習

このプロジェクトは古いコードであり、フローチャート、ビジネスフロー、データディクショナリがないため、すべてを最初からやり直す必要があります。

## 成果

現在、古いプログラミングの 80% が置き換えられていますが、モジュールが圧縮ファイルであるため、エンジニアとプロジェクト所有者を元に戻すことができず、一部のビジネス フローは移行できません。
